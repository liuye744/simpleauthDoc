##  特性
这是一款基于SpringBoot的轻量化渐进式的权限校验和访问控制的框架。适用于快速构建中小型项目。上手简单

[GitHub地址](https://github.com/liuye744/SimpleAuth)

## 适用场景
* 项目复杂度不高，权限校验简单
* 鉴权方案未固定，随时可能更改
* 快速构建RBAC(基于角色的访问控制)
* 复杂多样且非侵入式的参数校验
* 限制用户访问次数
* 小粒度的访问控制
* 简单的安全防护
